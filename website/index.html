ruby-dvi

Get Version

0.1.0

What

This is a library to read DVI and TFM file format.

Installing

sudo gem install dvi

API document

http://dvi.rubyforge.org/rdoc/

The basics

Read DVI file

Dvi.read(File.open(path))

Read TFM file

Dvi::Tfm.read(File.open(path))

Test

rake spec

You should run before test:

rake dvi:lsr

Forum

http://groups.google.com/group/ruby-dvi

How to submit patches

Read the 8 steps for fixing other people’s code and for section 8b: Submit patch to Google Groups, use the Google Group above.
